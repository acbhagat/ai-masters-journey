## 📌 About

This repository tracks my **AI learning journey from basics to advanced topics**, organized into progressive modules:

- **Fundamentals:** Programming logic, environment setup, Git workflow
- **Core & Advanced Python:** Data structures, functions, classes, and OOP
- **Data Handling:** Pandas, NumPy, visualization, and preprocessing
- **Machine Learning:** Regression, classification, clustering, evaluation metrics
- **Deep Learning:** Neural networks, CNNs, RNNs, PyTorch/TensorFlow
- **NLP:** Tokenization, embeddings, transformers, language models
- **AI Systems:** Docker, APIs, system design, workflow automation
- **Projects:** Real-world applications for portfolio building
- **Deployment & MLOps:** Production-ready pipelines, CI/CD, cloud deployment

This repo is **open for learning and collaboration**, showcasing a stepwise progression from learning fundamentals to building deployable AI systems.
